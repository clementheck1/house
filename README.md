# DR House Lifting - Professional Homepage

A modern, enterprise-level homepage for DR House Lifting, showcasing their professional house lifting services using advanced Failsafe Arrow technology.

## Features

- **Modern React Application**: Built with React 18 and Vite for optimal performance
- **Responsive Design**: Fully responsive layout that works on all devices
- **Interactive Animations**: Smooth scroll animations using Framer Motion
- **Professional Styling**: Tailwind CSS with custom brand colors (black and #20d9e3)
- **Lucide Icons**: Clean, professional icon set throughout

## Key Sections

1. **Hero Section**: Eye-catching hero with background image and gradient overlay
2. **Trust Indicators**: Statistics showcasing company credentials
3. **Services**: Comprehensive overview of all services offered
4. **Technology**: Detailed explanation of the Arrow lift system advantages
5. **Process**: Three-step project workflow (Planning → Permits → Lift Off)
6. **Benefits**: Six key advantages of house lifting
7. **Contact**: Contact form and company information

## Technology Stack

- React 18.2
- Vite 5.0
- Tailwind CSS 3.3
- Framer Motion 10.16
- Lucide React 0.294

## Getting Started

### Prerequisites

- Node.js 16+ installed
- npm or yarn package manager

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The site will be available at `http://localhost:3000`

### Build for Production

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```
